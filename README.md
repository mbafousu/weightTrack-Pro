# WeightTrack-Pro (Personal)

WeightTrack-Pro is a single-page personal weight tracking web application that allows users to log daily weight entries, set goals, and visualize progress over time.  
The project emphasizes DOM manipulation, event-driven programming, Browser Object Model (BOM) usage, and form validation, following real-world dashboard and enterprise-style UI patterns.

## Features

- Personal user profile (name, date of birth, automatic age calculation)
- Daily weight logging with optional notes
- Goal weight tracking
- Progress dashboard with KPI cards
- Weight history table with edit and delete actions
- Unit switching (lbs â†” kg) with automatic conversion
- Persistent data storage using `localStorage`
- Export weight history to CSV
- Print-friendly view
- Responsive UI built with Bootstrap 5

## Problem This Project Solves

Many individuals track weight inconsistently using notes, spreadsheets, or multiple apps.  
WeightTrack-Pro provides a clean and structured solution to:

- Track daily weight consistently
- Monitor progress toward personal goals
- Detect sudden weight changes
- Store data locally without requiring accounts
